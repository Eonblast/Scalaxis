1) download new version
2) extract somewhere else
3) copy ebin/, include/, src/, LICENSE, vsn.mk to <scalaris>/contrib/yaws
4) call '(cd contrib; patch -p0 < yaws_json_encode_lists_patch.diff; cd -)'
5) call '(cd contrib; patch -p2 < yaws_json_encode_floats.diff; cd -)'
6) configure && make in the extracted yaws dir
7) copy src/mime_types.erl and src/yaws_generated.erl to their respective folders in <scalaris>/contrib/yaws
8) create <scalaris>/contrib/yaws/src/yaws_configure.hrl with content:
%-define(HAVE_SENDFILE, true).
9) that's it, go back to <scalaris>/ and run ./configure && make
